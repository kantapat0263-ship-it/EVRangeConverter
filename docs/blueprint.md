# **App Name**: EV Range Converter

## Core Features:

- Standard Selection Tabs: Navigate between CLTC, WLTP, EPA, and NEDC standards to define the input unit, updating the input field label dynamically.
- Real-time Range Input: Enter electric vehicle range in kilometers; the system will instantly process and display conversions as you type.
- Dynamic Conversion Engine: Accurately apply conversion formulas to transform input range from any standard (CLTC, WLTP, EPA, NEDC) into all other standards.
- Converted Results Display: Show calculated equivalent ranges in distinct cards for WLTP, EPA, NEDC, with a highlight for the user's original input value.
- Responsive Layout: Ensure the application UI is fully functional and visually appealing across various devices, from mobile to desktop.
- AI-Powered Context Tool: An AI-powered tool to provide concise explanations for each range standard and contextual details about their differences, aiding user understanding.

## Style Guidelines:

- Primary accent color: A vibrant neon blue (#33BCFF) evoking technological advancement and electricity, standing out against dark backgrounds.
- Background color: A deep, almost black, subtly bluish grey (#121516), providing a sophisticated dark mode canvas for glowing elements.
- Secondary accent color: A bright, slightly greenish-cyan (#75F0DB), analogous to the primary blue, used to create dynamic contrasts and interactive element states.
- Headline and body font: 'Space Grotesk' (sans-serif) for a modern, technical, and clean aesthetic, suitable for both headlines and concise information displays.
- Use sleek, minimalist outline icons that align with a futuristic and technological theme, complementing the glow effects.
- Utilize Glassmorphism for elements like result cards and input fields, featuring blurred transparency to give depth. Incorporate subtle 'glow' effects around interactive elements and focus states, especially for the active tab and result highlights.
- Implement smooth, subtle transitions for input changes, result updates, and tab selections. Ensure numbers in the result display animate fluidly rather than appearing instantaneously, enhancing user experience.
